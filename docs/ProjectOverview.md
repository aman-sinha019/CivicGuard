# Project Overview

## Project Name
CivicGuard

## Description
CivicGuard is an AI-powered civic safety platform that enables citizens to report incidents, locate nearby emergency services, and access an AI-powered dashboard for analytics. The platform aims to improve communication between citizens and authorities.

## Objectives
- Enable easy incident reporting
- Provide location-based emergency assistance
- Visualize reports through an AI dashboard
- Improve public safety using technology

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Google Maps API
- Python (AI)
