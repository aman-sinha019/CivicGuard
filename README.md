# 🛡️ Civic Guard

**AI-Powered Smart Incident Reporting & Emergency Response System**

## 📖 Overview

**Civic Guard** is a smart web-based platform designed to improve public safety by enabling citizens to report incidents in real time. The platform uses location-based services, AI-assisted incident categorization, and an administrative dashboard to help authorities receive, verify, and manage reports efficiently.

The goal is to reduce emergency response time, improve communication between citizens and authorities, and build a safer, more responsive community.
